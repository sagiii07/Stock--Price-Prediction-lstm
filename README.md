 # Stock Price Prediction using LSTM

This project predicts TCS stock prices using LSTM Neural Networks.

## Features
- Yahoo Finance Data
- MA50 & MA100 Indicators
- Daily Returns
- LSTM Deep Learning Model
- Next Day Prediction
- Interactive Dashboard

## Technologies
- Python
- TensorFlow
- Pandas
- NumPy
- Plotly
- Scikit-Learn

## Results
Interactive stock prediction dashboard with actual vs predicted visualization.
